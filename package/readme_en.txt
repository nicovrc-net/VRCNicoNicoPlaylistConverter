# VRCNicoNicoPlaylistConverter
This is almost the same content as the booth description, but with some additional notes for people who read this file.

## What is this
A tool that converts playlists from Nico Nico Douga into the playlist format used by VRC video players.

## How to use
### Start
On Windows, double-click the `start(.bat)` file.
On Linux, make `start_forLinux.sh` executable and run it.

### Login
Enter your Nico Nico email and password to log in.
* It supports two-factor authentication, but don’t use your main account with a sketchy tool like this.

### Output
Follow the on-screen instructions to enter information and make selections.


## Verified video players
- iwaSync (v3.7.1 and later)
- Kinel式 (りら式) (v2.5.5 and later)
- YamaPlayer (v1.5.16 and later)
- VizVid (v1.4.11 and later)
- Ayaplayer2 (β1.0.0 and later)

The following are supported as bonus features:
- Sliden
- UnaSlides
